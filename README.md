# 🏪 shopeee - 店舗管理AIアシスト

飲食・小売店舗のための資材ECプラットフォーム。AIエージェントが店舗運営をサポートします。

## 機能

### EC機能
- 商品カタログ（カテゴリ検索・フィルタ）
- 商品詳細ページ
- カート機能（追加・数量変更・削除）
- チェックアウト

### 管理者ダッシュボード
- KPIダッシュボード（売上・受注・顧客・在庫）
- 受注管理（ステータス管理）
- 商品管理
- 顧客管理（ティア・LTV分析）
- 在庫管理（アラート・バー表示）
- 分析レポート
- 設定画面

### AIエージェント
- Claude API連携のチャットボット
- ストアデータ連携による文脈理解
- 在庫提案・売上分析・発注推奨
- クイックアクションボタン

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build
```

## デプロイ（Vercel）

1. GitHubにリポジトリをプッシュ
2. [Vercel](https://vercel.com) でインポート
3. フレームワーク: **Vite** を選択
4. デプロイ

## 技術スタック

- **React 18** + **Vite**
- **Tailwind CSS**
- **Claude API** (AIエージェント)
- **Vercel** (デプロイ)
